# Support

For setup help, open a GitHub issue with:

- operating system;
- Node version;
- command you ran;
- full error output;
- whether you are running local sample data or devnet proof flows.

For security-sensitive reports, use `SECURITY.md` instead of a public issue.

Useful commands:

```bash
cd nagarik-signal
npm run seed:demo
npm run test:unit
npm run verify:data
npm run typecheck
npm run lint
npm run build
npm run final:preflight
```
