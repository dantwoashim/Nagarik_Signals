# Demo Script

## 0:00-0:15 - Hook

Nepal does not lack complaint channels. It lacks public memory for ignored
civic issues.

## 0:15-0:30 - Product

Nagarik Signal creates public proof objects for public problems. Janamat shows
what citizens think. Nagarik Signal shows what citizens proved.

## 0:30-1:00 - Report

Submit a safe public infrastructure issue with photo, category, ward,
approximate location, and first-observed date. The app hashes the evidence,
canonical metadata, and location commitment.

## 1:00-1:35 - Issue Page

Open the public issue page. Show evidence, status, Days Ignored, verification
count, metadata hash, evidence hash, location hash, and Explorer link.

## 1:35-2:05 - Verify

A second citizen verifies once. Duplicate verification is rejected by PDA design.

## 2:05-2:35 - ProofPanel

Recompute the displayed metadata hash and compare it against the on-chain issue
state.

## 2:35-2:50 - Steward Resolution

Open the steward console. Filter the queue to indexed devnet issues, move the
issue to in progress, then resolve it with an after-photo. Show the StatusUpdate
transaction and the resolved timeline entry.

## 2:50-3:00 - Dashboard

Show unresolved issues, average Days Ignored, ward leaderboard, resolved proof,
and the frozen "Resolved after X days" counter.

Final line:

Janamat shows what citizens think. Nagarik Signal shows what citizens proved.
