# Competitors

| Product | What it does | Nagarik Signal wedge |
|---|---|---|
| Hello Sarkar | Government grievance intake | Nagarik creates independent public proof. |
| FixMyStreet | Public issue reporting | Nagarik adds Solana proof and timeline verification. |
| SeeClickFix | Municipal service requests | Nagarik can work before municipal adoption. |
| Ushahidi | Crowdsourced maps | Nagarik focuses on issue proof and status history. |
| Janamat | Civic opinion and petitions | Nagarik focuses on evidence citizens proved. |

Nagarik Signal does not replace these systems. It creates a public memory layer
they can reference later.
