# Phase 6 Submission Readiness

Status: code and artifact gate for final civic hackathon submission.

## Required Claim

Civic Impact Markets is a verified civic forecasting and sponsor-funded action reward prototype. The Solana proof is a devnet receipt settlement bundle with replay protection, public artifacts, SDK verification, and a Phase 5 conviction signal spec/artifact.

Do not claim:

- live municipal data integration
- live private identity proof
- real-value mainnet readiness
- physical repair completion proof
- forecast-linked reward payout

## Fresh Clone Verification

Evaluator command path:

```bash
npm ci
npm run phase6:final-checks
npm run phase6:browser-readiness
npm run phase6:assert-submission
```

This is the fresh-clone contract for judges. A local uncommitted working tree cannot honestly be called a fresh clone until it is committed and tagged, so the final personal step is to run the same commands from a clean clone of the tagged commit before submission.

## Accessibility Verification

The browser readiness script fails if:

- horizontal overflow exists
- no focusable controls exist
- controls lack accessible names
- inputs lack labels
- images lack `alt`
- pages have no `h1`
- reduced-motion emulation is not active

This is not a replacement for a full manual audit, but it is the enforced Phase 6 automated gate.
